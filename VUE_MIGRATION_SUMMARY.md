## 🚀 快速开始

### 方式1：一键启动（推荐）

**Windows:**
```bash
双击运行 start_game.bat
```

**Linux/Mac:**
```bash
./start_game.sh
```

### 方式2：手动启动

**后端：**
```bash
python texas_holdem_backend.py
```

**前端：**
```bash
cd frontend
npm install
npm run dev
```

访问：http://127.0.0.1:3000

## 🎮 游戏特性

### 座位布局
- **下方（座位0）**：人类玩家（你）
- **左侧（座位1）**：机器人1
- **右侧（座位2）**：机器人2（庄家）

### 操作按钮
- 弃牌 / 过牌 / 跟注 / 下注 / 加注

### AI策略
机器人根据手牌强度自动决策：
- **强牌**（>70%）：加注或跟注
- **中等牌**（40-70%）：跟注或过牌
- **弱牌**（<40%）：弃牌或过牌
