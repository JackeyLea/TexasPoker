# 德州扑克 - Vue前端

这是一个使用Vue 3开发的德州扑克游戏前端界面，与后端Flask API进行通信。

## 项目结构

```
frontend/
├── index.html              # HTML入口文件
├── package.json            # 项目依赖配置
├── vite.config.js          # Vite构建配置
├── src/
│   ├── main.js             # Vue应用入口
│   ├── App.vue             # 根组件
│   ├── style.css           # 全局样式
│   └── views/
│       └── GameView.vue    # 游戏主界面组件
└── README.md               # 说明文档
```

## 功能特性

- ✅ 完整的德州扑克游戏流程（翻牌前、翻牌、转牌、河牌、摊牌）
- ✅ 1个人类玩家 + 2个机器人玩家
- ✅ 机器人AI自动决策
- ✅ 实时游戏状态更新
- ✅ 手牌牌型判断（皇家同花顺、同花顺、四条等）
- ✅ 胜负判断和筹码分配
- ✅ 响应式设计，支持移动端
- ✅ 美观的牌桌界面

## 技术栈

- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **Axios** - HTTP客户端
- **Flask** - 后端API（原有项目）

## 安装和运行

### 前置要求

- Node.js 16+ 
- Python 3.8+（用于运行后端）
- 后端Flask服务已启动

### 安装步骤

1. **安装依赖**
```bash
cd frontend
npm install
```

2. **启动后端服务器**（在另一个终端）
```bash
python texas_holdem_backend.py
```

后端服务将运行在 http://127.0.0.1:5000

3. **启动前端开发服务器**
```bash
npm run dev
```

前端服务将运行在 http://127.0.0.1:3000

4. **构建生产版本**
```bash
npm run build
```

## 游戏说明

### 座位布局
- **座位0**（下方）：人类玩家（你）
- **座位1**（左侧）：机器人1
- **座位2**（右侧）：机器人2（庄家位）

### 操作顺序（顺时针）
右侧机器人 → 下方人类 → 左侧机器人

### 操作按钮
- **弃牌**：放弃本轮游戏
- **过牌**：不下注，跳过回合（无人下注时可用）
- **跟注**：跟上其他玩家的下注
- **下注**：主动下注
- **加注**：提高当前下注额

### 游戏规则
- 起始筹码：1000
- 小盲注：10
- 大盲注：20
- 使用标准德州扑克规则

## API接口

前端通过以下API与后端通信：

- `GET /api/game/state` - 获取游戏状态
- `POST /api/game/start` - 开始新游戏
- `POST /api/game/action` - 玩家执行动作
- `POST /api/reset` - 重置游戏

## 迁移说明

本项目从纯HTML/JavaScript版本迁移到Vue.js版本，主要改进：

1. **组件化**：将游戏逻辑拆分为Vue组件
2. **响应式数据**：使用Vue的响应式系统自动更新UI
3. **更好的状态管理**：集中管理游戏状态
4. **开发体验**：热重载、更好的调试工具
5. **构建优化**：使用Vite进行构建和打包
