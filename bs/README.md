# TexasPoker

B/S架构的德州扑克

## 结构

### 前端

一个独立的 HTML 页面，包含游戏界面，并通过 Fetch API 与后端通信。

### 后端

一个基于 Flask 的 Python 服务器，负责管理游戏状态、逻辑判断，并提供 RESTful API。

实现了扑克牌、玩家、游戏状态等完整的数据模型。包含了发牌、下注轮次管理、玩家动作处理、游戏阶段推进、扑克手牌排名算法、牌力比较和胜者判定逻辑等核心逻辑。

## 环境

- python 3.14.3
- Windows 10 Pro 22H2 19045.6466

## 运行

```bash
# 后端
python texas_holdem_backend.py

# 前端 双击打开index.html
```
