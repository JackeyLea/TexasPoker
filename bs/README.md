# TexasPoker

B/S架构的德州扑克

python texas_holdem_backend.py

http://127.0.0.1:5000


## 结构

### 前端

一个独立的 HTML 页面，包含游戏界面，并通过 Fetch API 与后端通信。

动态渲染游戏界面，并将玩家的点击动作转换为API调用发送给后端。

### 后端

一个基于 Flask 的 Python 服务器，负责管理游戏状态、逻辑判断，并提供 RESTful API。

实现了扑克牌、玩家、游戏状态等完整的数据模型。包含了发牌、下注轮次管理、玩家动作处理、游戏阶段推进等核心逻辑。

扑克手牌排名算法、牌力比较和胜者判定逻辑

## 环境

- python 3.7+
- python 3.14.3
